# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data to uncover business trends, monitor KPIs, and support data-driven decision-making. The analysis was performed using Excel to transform raw sales records into actionable insights.

## Objectives
- Evaluate overall sales performance
- Track key business KPIs such as revenue, profit, and sales trends
- Identify top-performing products and categories
- Discover seasonal patterns in sales

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning Techniques

## Key Insights
- Analyzed 5,000+ sales records
- Built dashboards tracking 10+ KPIs
- Identified revenue-driving categories
- Detected seasonal sales peaks

## Business Impact
This project demonstrates how Excel-based analysis can help businesses improve product strategy and make informed decisions.
